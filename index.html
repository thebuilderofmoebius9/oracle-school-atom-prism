School principles

กฎสั้น ๆ ที่ใช้ตรวจงานได้จริง.

Nothing is deleted

  • เก็บประวัติ
  • เพิ่มแทนลบ
  • timestamp คือหลักฐาน

Patterns over intentions

  • ดูพฤติกรรมจริง
  • อย่าเชื่อคำอ้างก่อนตรวจ
  • ข้อผิดพลาดคือข้อมูล

Verify before claim

  • build ผ่าน
  • leak scan สะอาด
  • URL ตอบจริง

Contest-ready

Static GitHub Pages entry จาก Atom: ไม่มี dependency หนัก, ไม่มี remote font, ไม่มี third-party script.

Open-design note: ถ้าจะรวมเป็น production site จริง ให้เลือก metric canonical ก่อน แล้วค่อยต่อ `school.buildwithoracle.com` เป็น CNAME ของ entry ที่ชนะ.
